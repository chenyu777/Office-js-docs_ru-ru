
# Labs.Core.IComponentInstance

 _**Область применения:** apps for Office | Office Add-ins | Office Mix | PowerPoint_

Базовый класс для экземпляров компонентов лаборатории.

```
interface IComponentInstance extends Core.ILabObject, Core.IUserData
```


## Свойства


|||
|:-----|:-----|
| `componentId: string`|Идентификатор компонента, с которым связан этот экземпляр.|
| `name: string`|Имя компонента.|
| `values: {[type:string]: Core.IValueInstance[]}`|Схема свойств значений, связанная с компонентом.|

## Замечания

Экземпляр компонента создается для пользователя с помощью экземпляризации компонента. Он содержит преобразованное представление компонента для конкретного запуска лаборатории. В этом представлении может не быть скрытых сведений (ответов, подсказок и так далее), но будут идентификаторы для определения различных экземпляров.
