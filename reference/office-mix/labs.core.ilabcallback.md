
# Labs.Core.ILabCallback

 _**Область применения:** apps for Office | Office Add-ins | Office Mix | PowerPoint_

Интерфейс для обработки методов обратного вызова Labs.js.

```
interface ILabCallback<T>
```


## Сигнатура обратного вызова

 `(err: any, data: T): void`

 **Параметры обратного вызова**


|||
|:-----|:-----|
| _err_|Возвращает значение **Null** при отсутствии ошибок. Возвращает другое значение (не **Null**) при возникновении ошибки.|
| _data_|Данные, возвращенные при обратном вызове.|
