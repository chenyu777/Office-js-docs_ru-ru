
# Labs.getTimeline

 _**Область применения:** apps for Office | Office Add-ins | Office Mix | PowerPoint_

```
function getTimeline(): Timeline
```


## Возвращаемое значение

Возвращает объект [Labs.Timeline](../../reference/office-mix/labs.timeline.md).
