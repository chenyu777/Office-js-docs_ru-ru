
# Элемент DesktopSettings
Задает исходное расположение и параметры элемента управления, которые применяются при использовании почтовой надстройки на настольном компьютере.

 **Тип надстройки:** почтовые.


## Синтаксис:


```XML
<DesktopSettings>
   ...
</DesktopSettings>
```
