
# Элемент Dictionary
Определяет параметры для надстройки области задач, которая обеспечивает поддержку дополнительного словаря.

 **Тип надстройки:** надстройки области задач.


## Синтаксис:


```XML
<Dictionary>
   ...
</Dictionary>
```
