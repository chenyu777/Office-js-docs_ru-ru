
# Элемент TabletSettingst
Задает параметры управления, которые применяются при использовании вашей почтовой надстройки на планшете.

 **Тип надстройки:** почтовые.


## Синтаксис:


```XML
<TabletSettings>
   ...
</TabletSettings>
```
